•	StrategyContext 파일은 React 애플리케이션에서 상태 관리를 위한 컨텍스트를 정의합니다.
•	주로 상태를 생성하고 관리하며, 이 상태를 여러 컴포넌트에서 공유하고 사용할 수 있도록 합니다.
•	전역적인 상태 관리를 통해 애플리케이션의 데이터 흐름을 제어하는 데 사용됩니다.

•	상태 관리: useState나 useReducer를 통해 상태를 생성하고 관리합니다. 이 상태는 여러 컴포넌트에서 공유될 수 있습니다.
•	전역 데이터 공유: 컨텍스트를 사용하여 전역 상태를 관리하고, 필요한 경우 해당 상태를 하위 컴포넌트에 제공합니다.
•	애플리케이션 데이터 흐름 제어: 여러 컴포넌트 간에 데이터 흐름을 쉽게 제어하고 관리할 수 있도록 합니다.
